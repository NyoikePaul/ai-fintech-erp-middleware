# Contributing to AI-Fintech ERP Middleware

We welcome contributions! To maintain senior-level code quality:

1.  **Branching:** Create your feature branch from `main`.
2.  **Standards:** Follow PEP 8 and ensure all Pydantic models are fully typed.
3.  **Testing:** Add a pytest for any new service or domain logic.
4.  **Pull Requests:** Provide a clear description of changes and link any related issues.

Together, we're building the future of African Enterprise Tech.
